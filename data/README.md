# Data Files

This directory contains split metadata and small release artifacts only.

- `splits/egteagaze/`: EGTEA Gaze+ train/test split files used by configs.
- `splits/ego4d/`: Ego4D stride-8 train/test window split files used by configs.
- `egoexo4d_splits/split_assignments.csv`: EgoExo4D benchmark split assignments included with this release.

Raw frames, videos, official annotations, and generated frame caches are not included.
